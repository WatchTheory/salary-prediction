


# Salary Prediction Analysis

Predicting Data Analyst Salaries from a fake dataset to determine appropriate data analyst salaries.


## About This Project

This study presents a comprehensive salary prediction analysis utilizing machine learning techniques to forecast individual compensation for Data Analyst based on fake data location, professional, and employment factors. Accurate salary prediction models are valuable for job seekers, employers, human resource departments, and policymakers seeking to understand compensation trends and reduce salary disparities in the labor market.


> [!WARNING]
The `messy_data_analyst_job.csv` dataset is fake data created by me only for this project.
